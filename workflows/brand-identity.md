# Workflow: Brand Identity

## Goal

Turn a client's positioning into a coherent visual identity system — naming (where needed), logo/wordmark direction, colour palette, typography, and brand guidelines — that the rest of the team (ads, landing pages, content) can apply consistently. The output is a reviewable identity direction; a human approves before it is used.

## When to use

A new or rebranding client needs a brand identity built from the ground up, or an existing brand needs a documented system so every channel looks consistent. (For applying an already-defined brand to one specific asset, use the relevant channel workflow instead.)

## Steps

1. Review the client context, market, and audience (`clients/<client>.md`), with the Belgian/NL market context (`knowledge/market-competitive-research.md`) where relevant.
2. Clarify positioning: what the brand stands for, who it is for, and how it should feel (the adjectives), plus any existing assets or constraints.
3. Where naming is in scope, propose name directions with rationale; otherwise work from the existing name.
4. Define the visual system: logo/wordmark direction, colour palette (with usage), typography, and imagery/iconography style.
5. Set the brand voice in step with `knowledge/agency-foundations.md` so the verbal and visual identity match.
6. Write concise brand guidelines: do's and don'ts, spacing/usage rules, and examples, so other agents apply it consistently.
7. Prepare the human approval summary; flag what needs client input (final name, legal/trademark checks, asset production).

## Agents involved

- Orchestrator Agent (routes and briefs)
- Brand & Identity Designer (lead — identity system)
- Copywriter (naming and verbal identity, where needed)
- Creative Designer (visual exploration and asset production, where needed)

## Required output

Follow `templates/task-output.md` and `knowledge/agency-foundations.md`. Must include:

- Positioning recap (what the brand stands for, for whom, how it should feel)
- Name direction(s) with rationale (where naming is in scope)
- Visual system (logo/wordmark direction, colour palette, typography, imagery)
- Brand voice (in step with the tone of voice)
- Brand guidelines (do's, don'ts, usage)
- Open questions / what needs client input (final name, trademark, assets)
- Human approval required before use
